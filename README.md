# ✨ אסתי אירועים

אתר לנדינג פייג׳ לעסק של אסתי — עמדות מזון, ציורי פנים, סדנת תכשיטים ועמדות צילום לאירועים.

🌐 **האתר החי:** https://shlomialon.github.io/esti-events

---

## הגדרת GitHub Pages

1. כנסי להגדרות הריפו: **Settings → Pages**
2. תחת **Source** בחרי: `Deploy from a branch`
3. בחרי branch: `main` ותיקייה: `/ (root)`
4. לחצי **Save**
5. תוך כ-2 דקות האתר יהיה חי! 🎉

---

## הגדרת טופס צור קשר (Formspree — חינם)

1. כנסי לאתר: https://formspree.io
2. צרי חשבון חינם עם האימייל שלך
3. צרי form חדש (New Form)
4. העתיקי את **Form ID** (נראה כך: `xpzgkwqr`)
5. פתחי את `index.html` וחפשי:
   ```
   YOUR_FORM_ID
   ```
6. החליפי אותו ב-ID שקיבלת מ-Formspree
7. שמרי ועלי מחדש לגיטהאב
